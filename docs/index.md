# Sortie

Sortie turns issue tracker tickets into autonomous coding agent sessions.

Single binary. Zero dependencies. SQLite persistence.

## What is Sortie?

Sortie is an orchestration service that continuously reads work from an
issue tracker (Jira, etc.), creates an isolated workspace for each issue,
and runs a coding agent session (Claude Code, etc.) for that issue inside
the workspace.
